@yakstagram 여드름약 아무거나 바르지 마세요.🙌🏻 *상태별 여드름약 정리* 🎾아프지는 않지만 알이 차려고 하는 상태 -> 파티마겔 균 성장 억제효과, 각질 녹여줌 자극성이 커서 극소량 바르고 처음에는 2일에 한번 바르다가 적응되면 1일에 1번 -> 1일 2회로 점진적으로 늘려주세요. ⚾빨갛게 알이 차고 아픈 상태 -> 큐아크네, 애크논 등 여드름 균 제거 + 염증 억제 기능 피부가 민감한 사람도 무난히 사용 가능 ️ ️🏐블랙헤드, 화이드헤드, 좁쌀여드름 -> 클리어틴 피지 녹이는 효과 자극적이면 2일에 1회 -> 1일 1회 -> 1일 2회로 점진적으로 적응해가기 ⚽️좁쌀, 화농성 모두 있는데 색소 침착 잘 생기는 사람 -> 아젤리아 여드름균 증식 억제 + 모공 막히지 않도록 하는 기능+색소침착 방지 효과 🏀여드름 짠 직후 감염 예방 -> 에스로반, 베아로반 모낭염에 잘 듣고, 추가 감염 예방에 좋은데 여드름과 모낭염은 육안 구분이 어렵긴 해요. 😓 *여드름이 잘 나는 분께 도움이 될만한 영양제* 💙오메가3 - 과도한 피지 분비 억제, 염증 억제 💙비타민 A D E 아연 -> 염증과 관련된 비타민. 종합비타민을 드세요. 🌞여드름약 사용은 세안과 기초 스킨 로션 후에 바르고, 약을 바른 후 자외선차단제를 꼭 발라주세요!! 🫣여드름약이 필요한 친구가 있다면 공유 🤔궁금한 건 질문해주세요🙌🏻 #약사 #약국 #약스타그램 #여드름 #손댄스 ♬ 可愛くてごめん (feat. かぴ) - HoneyWorks
